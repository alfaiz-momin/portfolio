My workflow

Mobile First approach:

Nowadays most of the traffic is coming from small devices. Because of that developer create Mobile First design method.

In this method, we write CSS code for smaller devices first and then write media query for medium and larger devices.

By using this method we can improve our website user experience.

Image Optimization:

We all know Images consumes most of the downloaded bytes on a web page. And also occupy major visual space of the screen.

We can optimize our images and write some tricky code for saving data and screen spaces, especially on small screen.

We can speed up our web page by optimizing images and icons. And save users data at the same time.

File and code management:

I manage all files and related content (image, icon, etc.) in a proper convection of modern web development. I use modules method to keep all code files in proper way.

I write semantic code with comments. And I use BEM method for naming CSS classes (not all time).
